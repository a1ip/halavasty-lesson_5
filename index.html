Массивы

Массивы

Это объекты, которые хранят значения (любого типа) не в именованных свойствах/ключах, а в ячейках, доступных по числовому индексу.

JS используют 0 в качестве индекса первого элемента массива.

Именование переменные и методов

  • Идентификатор должен начинаться с a-z, A-Z, $ или _
  • Нельзя использовать "зарезервированные слова" - for, in, if
  • Нельзя использовать null, true и false
